Consistent and Readable
for Beginners

C+ gives a more readable, as well as consistent syntax for beginners. Some example include the "let" statement. While in C you must use items like "int" and "char," you can use a single "let" statement in C+.

C+
A language for all

C+'s compiler is written in Python, giving fast and easy developing experience. Also because of this, C+ is easy to modify and extend. Furthermore, writing it in Python lets the compiler run on any platform.

C+ main idea is to make C readable, consistent, and easy to learn for beginners. It accomplishes this by having a brand new syntax, but still with a one to one translation to C, making sure none of the power of C is lost.

C+ also has an easy setup script, written in both Python and Rust, the setup is powerful, fast, and extensive. The setup installs all dependencies and tools needed to compile and run C+ programs.

Why Use C+

C+ is good because it has...

Readability

C+ prioritizes readability for beginners, making it easier to understand and write code. This not only helps people to learn C+ easier, but also helps developing, debugging, and sharing code.

Consistency

C+ provides a consistent syntax that makes it easier to learn and use. This consistency helps reduce confusion and errors, especially for beginners. Some examples include the use of "let" instead of "int" and "char".

Power

C+ has one to one line translation with C, which not only allows for fast compile time but also lets programmers get the power of C, without the headaches of C.
